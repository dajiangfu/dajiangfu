## 🔊 Audio / Music System

### 🎧 播放与解码
- **播放器系统**：piCorePlayer
- **音频输出**：USB Audio
- **DAC**：山灵 UA6
- **解码架构**：CS43131 ×4（多 DAC 并联）

### 🎼 取向与原则
- 关注 **低噪声底、稳定时钟与供电完整性**
- 重视 **长期聆听的舒适度**，而非刺激感
- 测量数据是基础，但不是终点
- 声音应当自然、干净、不抢注意力

> 好的音频系统，  
> 不会提醒你“这是系统”，  
> 只会让你继续把音乐听下去。

## 🧠 Philosophy / 工程哲学

- 系统的复杂性应当 **被驯服，而不是被展示**
- 自动化不是为了偷懒，而是为了减少不可控变量
- 稳定不是“从不出错”，而是 **出错时行为可预期**
- 配置文件是系统的一部分，也是文档的一部分
- 真正成熟的系统，  
  往往看起来 **无聊、安静、缺乏存在感**

> 幻想并非逃避现实，  
> 而是为现实提供一个更高、更安静的参照系。  
>  
> 正因为现实永远不完美，  
> 才需要一个不喧哗的理想，  
> 来约束我们正在做的每一个选择。
